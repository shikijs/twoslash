TSX Twoslash Examples

With Twoslash:

tsx
import React from "react"


export const MyTitle = () => (
    <h1>Title</h1>
)

Without Twoslash:

tsx
import React from "react"
export const MyTitle = () => (
    <h1>Title</h1>
)
